# Документы

## Получение

Получить все документы с ценами

* **URL**

    `GET`  **/document**

* **Ответ**

    ```json
    {
      "status": "OK",
      "response": [
        {
          "id": 1,
          "name": "Гражданство РФ",
          "price": 40
        },
        {
          "id": 2,
          "name": "Загран паспорт РФ",
          "price": 30
        }
      ]
    }
    ```
